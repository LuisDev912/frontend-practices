# TO-DO List

A simple **to-do list project** built with **HTML, CSS, and JavaScript**.  
It demonstrates DOM manipulation, event handling, and dynamic styling.

---

## Project Structure

TO-DO/
│
├── index.html # Main page displaying the to-do list
├── styles.css # Styling for the page
├── script.js # Logic for adding, marking, and removing tasks
└── website-layout.png # Low-fidelity design mockup

---

## Features

- Preloaded tasks:
  - Task 1  
  - Task 2  
- Add new tasks using an input and "Add" button.  
- Mark tasks as completed → text becomes **italic** and **strikethrough**.  
- Remove all completed tasks with a dedicated button.  

---

## How to Use

1. Open `index.html` in your browser.  
2. To add a task:
   - Click the "Add new task to the list" button → input appears  
   - Type your task → click "Add"  
3. To mark a task as done → click on it; it will become italic and struck-through.  
4. To remove completed tasks → click the "Remove the selected tasks" button.  

---

## Requirements

- Modern web browser (no external libraries or frameworks used).
